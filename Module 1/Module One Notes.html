• Module 1: Introduction to Machine Learning
    • Definition: Machine learning is the subfield of computer science that gives “computers the ability to learn without being explicitly programmed.”
    • Images  -> Features Extraction -> A Machine Learning Model -> Predictions
    • Major machine learning techniques
      • Regression/Estimation – APredicting continuous values
      • Classification – Predicting the item class/category of a case
      • Clustering – Finding the structure of data; summarization
      • Associations – Associating frequent co-occurring items/events
    • Difference between artificial intelligence, machine learning and deep learning:
      • AI attempts to make computers intelligent to mimic the cognitive functions of humans. E.g. (Computer Vision, Language Processing, Creativity and Summarization)
      • Machine Learning covers the statistically component of AI. It teaches the computer to solve problems by looking at hundreds of thousands of examples.
      • Deep learning is a special field of machine learning where computers can make intelligent decisions on their own.

    • Supervised vs Unsupervised Learning
      • What is supervised learning?
        • We "teach the model" when with that knowledge it can predict unknown or future instances
        • We use data from a 'Labled dataet' to train the model.
        • The column headings in the labeled dataset represent attributes.
        • The columns are called features which include the data for each attribute
        • A row is called an observation
        • !labeled_data_example
          • Attributes -> UserID, Gender, Age, Salary, Purchased, Temperature, ...
          • Features -> Age, 19,35,... (essentiially that column)
          • Observation -> A row from that data
        • Data can be of two kinds: Numerical or Categorical
          • Numerical is just data which is numbers
          • Categorial is data which non-numerical
      • Types of Supervised Learning
        • Classification - The process of prediction a discrete class label, or category
        • Regression - The process of predicting continous values
      • What is Unsupervised learning?
        • In unsupervised learning we let the model work on its own to discover information that may not be visible to the human eye.
        • We give it some input data but don't have any output data to train the model with.
        • Clustering is a type of unsupervised learning technique
          • Clustering is grouping of data points or objjects that are somehow similiar by: 1. Discovering Structure, 2. Summarization, 3. Anomaly detection.
    • SUMMARY
      • Supervised Learning
        • Classification: Classifies labeled data
        • Regression : Predicts trends using previous leveled data
        • Has more evaluation methods than unspuervised learning
        • Controlled Environment
      • Unsupervised Learning
        • Clustering: Finds patterns and grouping from unlabeled data
        • Has fewer evaluation methods than supervised learning
        • Less controlled environment
